# csv_subtitles
CMAの出力のcsvから字幕表示用のファイルを生成します。
tt形式の字幕を作成します。

# 環境
Python3.7
jupyter-lab

# 使い方
1. 
```
mkdir data 
```
2. 
```
mkdir result
```
3. 必要なcsvファイルを`./data`に移動

4. 
```
python3 main.py
```

# データについて
下記の形式のcsvファイル
| timestamp | 発表者 | 発話内容 |
| ---- | ---- | ---- |
| 00:00:00 | speaker1 | こんにちは。 |
| 00:00:00 | speaker1 | 発表を開始します。 |


# 時間の算出についてメモ
CMAの仕様により、n秒のどこで発話されたかが記載されていない
同一分の文字列長を確認し、合計文字数の割合によって時間を推定する
